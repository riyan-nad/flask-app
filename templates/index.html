🔧 DevOps-Powered Flask Web Application

This project demonstrates the synergy between Flask and modern DevOps practices.

Key DevOps concepts implemented in this application include:

  • 🐳 Containerized with Docker for consistent environments
  • 🔁 CI/CD pipeline using GitHub Actions to automate builds and deployment
  • 📦 Image pushed to Docker Hub on every commit to main
  • 🗃️ SQLite used as a lightweight database for demonstration
  • 💡 Production-ready Dockerfile using python:3.11-slim
  • 🧪 Structured for future integration of testing and monitoring tools

Whether you're managing infrastructure or deploying apps, this project reflects real-world DevOps principles in a simple, understandable way.

Curious about the setup? Explore the Dockerfile, docker-compose.yml, and .github/workflows directory to see how it all comes together.
